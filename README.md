Python API and application for the Foscam FI8918W.

_Working_
* Connect to camera by URL
* Query camera status
* Capture image from camera
* Toggle IR on/off
* Pan/tilt camera
* Set and go to preset positions
* Patrol mode
* Authenticates through digest auth (Fixed #4)
